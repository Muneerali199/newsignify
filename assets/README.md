# Asset Files for Sign Language App

This directory contains the app icons and splash screens. The SVG files have been created for the app assets.

## Files Created:
- icon.svg - App icon (1024x1024)
- splash.svg - Splash screen (1284x2778) 
- adaptive-icon.svg - Android adaptive icon (108x108)

## To Use:
These SVG files need to be converted to PNG format for React Native. You can:
1. Use online converters to convert SVG to PNG
2. Use design tools like Figma, Sketch, or Adobe Illustrator
3. Use command line tools

## Recommended PNG Sizes:
- icon.png: 1024x1024 pixels
- splash.png: 1284x2778 pixels (or 1242x2688 for iPhone)
- adaptive-icon.png: 108x108 pixels

## Alternative:
You can also update app.json to reference the SVG files directly if your build process supports it.
